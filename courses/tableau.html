What You'll Learn

Eight modules, eight weeks — every topic taught with a working dataset and a real dashboard.

  • Tableau Desktop tour: workspace, shelves, marks card
  • Connecting to Excel, CSV and SQL data sources
  • Dimensions vs measures — the mental model
  • Building bar, line and pie charts from scratch
  • Saving, sharing and exporting your first dashboard
  • Calculated fields and table calculations
  • Parameters for interactive dashboards
  • Sets, groups and hierarchies
  • LOD expressions: FIXED, INCLUDE, EXCLUDE
  • Maps, dual-axis and Pareto charts
  • Custom shapes and color palettes
  • Layout containers and tiled vs floating elements
  • Filter actions, highlight actions, URL actions
  • Designing for executive vs analyst audiences
  • Mobile-first and responsive dashboards
  • Storytelling with Story Points
  • Tableau Prep Builder: flows, steps and outputs
  • Joins vs unions vs blends — when to use each
  • Pivoting wide and tall data
  • Cleaning text fields and handling nulls
  • Scheduling Prep flows for daily refresh
  • Live vs extract connections
  • Filtering at the source vs in Tableau
  • Reducing the number of marks and queries
  • Performance recorder and bottleneck analysis
  • Best practices for production dashboards
  • Tableau Server vs Tableau Cloud
  • Sites, projects, permissions and user roles
  • Scheduling extract refreshes
  • Subscriptions, alerts and embedded views
  • Row-level security for sensitive data
  • Project 1: e-commerce KPI dashboard
  • Project 2: HR attrition analysis
  • Project 3: financial performance scorecard
  • Code reviews and trainer feedback
  • Publishing to Tableau Public for your portfolio
  • Top 50 Tableau interview questions, with answers
  • Live SQL + Tableau combined assignments
  • Resume review, one-on-one
  • Mock interviews with hiring panelists
  • LinkedIn profile and portfolio polish
