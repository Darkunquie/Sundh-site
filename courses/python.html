Curriculum

What You'll Learn

Eight modules, eight weeks — every topic taught with code you write yourself, line by line.

  • Setup: Python 3, VS Code, virtualenvs
  • Variables, types and basic I/O
  • If/elif/else, loops and ranges
  • Strings, slicing and formatting
  • Your first script that solves a real problem
  • Lists: indexing, slicing, mutation
  • Dictionaries and key-value workflows
  • Sets and tuples — the underused pair
  • Comprehensions for one-line transforms
  • Choosing the right structure for the job
  • Functions, default args, *args/**kwargs
  • Scope, closures and decorators (gentle intro)
  • Classes, instances and methods
  • Inheritance and composition
  • Writing readable, testable code
  • Reading and writing text and CSV files
  • Working with JSON
  • Modules, packages and imports
  • Standard library tour: os, sys, pathlib
  • Building your own reusable module
  • Try / except / finally patterns
  • Built-in exceptions and when each fires
  • Raising and creating custom exceptions
  • Logging vs printing — production hygiene
  • Debugging with breakpoints in VS Code
  • HTTP basics: GET, POST, status codes
  • The requests library, end to end
  • Parsing JSON responses
  • API authentication: keys, tokens, OAuth basics
  • Building a small CLI that hits a real API
  • NumPy arrays vs Python lists
  • Pandas DataFrames in 30 minutes
  • Reading CSVs and quick analysis
  • Basic plotting with Matplotlib
  • Where to go next: Data Science, ML, AI
  • Project 1: a CLI utility (file organiser or to-do)
  • Project 2: an API-driven mini app
  • Project 3: a data analysis notebook
  • Top 50 Python interview questions, drilled
  • Resume polish and mock interviews
