Curriculum

What You'll Learn

Eight modules, ten weeks — every topic taught against a live Azure subscription with real datasets.

  • Azure portal, CLI and PowerShell tour
  • Subscriptions, resource groups, RBAC
  • Regions, availability zones and pricing tiers
  • Azure Active Directory and identity
  • AZ-900 concepts mapped to data work
  • Blob, File, Queue and Table storage
  • ADLS Gen2: hierarchical namespace and ACLs
  • Hot, cool, archive — choosing the right tier
  • Partitioning strategies for Parquet at scale
  • Lifecycle policies and cost optimization
  • Linked services, datasets and pipelines
  • Copy activity, mapping data flows
  • Triggers: schedule, tumbling window, event
  • Parameterization and pipeline reuse
  • CI/CD with ADF and Azure DevOps
  • Synapse workspace, pools and pipelines
  • Dedicated SQL pool: distributions and indexes
  • Serverless SQL over data lake files
  • Synapse Notebooks with Spark
  • Query tuning and performance patterns
  • Databricks workspace and clusters
  • PySpark essentials: DataFrames, joins, UDFs
  • Delta Lake: ACID on data lakes
  • Notebook orchestration with workflows
  • MLflow tracking for data science workloads
  • Event Hubs vs Service Bus vs IoT Hub
  • Stream Analytics queries and windows
  • Hot path vs cold path architectures
  • Output sinks: Power BI, ADLS, SQL
  • Building a real-time KPI dashboard
  • Managed identities and Key Vault
  • Private endpoints and VNet integration
  • Azure Purview: catalog, lineage, classification
  • Row-level and column-level security
  • Cost monitoring and budget alerts
  • DP-203 domain-by-domain breakdown
  • Mock exams with timed simulations
  • Capstone: end-to-end ETL pipeline (raw → curated → BI)
  • Code reviews and trainer feedback
  • Resume rebuild and mock interviews
