# Sunadh Technologies — Marketing Website

Career-first IT training institute in Ameerpet, Hyderabad. Static marketing site with dark premium theme.

## Live Site

**Domain:** [sunadhtechnologies.in](https://sunadhtechnologies.in)

## Pages

| Page | File |
|------|------|
| Home | `index.html` |
| Courses | `courses.html` |
| About | `about.html` |
| Admissions | `admissions.html` |
| Placements | `placements.html` |
| Course Details | `courses/*.html` (6 pages) |

## Tech Stack

- HTML / CSS / Vanilla JS
- Poppins + Inter (Google Fonts)
- Custom SVG icons
- AI-generated course hero images

## Design

- Dark theme (`#121412`)
- Glass morphism nav
- Orange `#FF6B1A` + Gold `#FFB627` accents
- Responsive (1024 / 768 / 480 breakpoints)

## Structure

```
├── index.html
├── about.html
├── admissions.html
├── courses.html
├── placements.html
├── colors_and_type.css
├── assets/
├── courses/
├── js/
└── styles/
```

## Run Locally

```bash
npx http-server . -p 8080
```

## Deploy

Static site. Upload to `public_html` on Hostinger. No build step.
